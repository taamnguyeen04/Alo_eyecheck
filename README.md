# Alo_eyecheck

# Các file
- processing.py - Logic tính toán Snellen và kiểm tra câu trả lời
- chatbot.py - Tích hợp Gemini AI để hướng dẫn, phản hồi động, và tư vấn
- main.py - Chương trình chính với flow hoàn chỉnh
- requirements.txt - Danh sách thư viện cần cài

# Cách chạy:
```
# Cài đặt thư viện
pip install -r requirements.txt

# Chạy chương trình
python main.py
```

# Lưu ý:
- Đảm bảo file .env có API key Gemini: GOOGLE_API_KEY=your_api_key_here
